How To Use This Application

To detect a tumor in an MRI scan, upload your MRI scan. After a few seconds, this should bring you to a new tab with the tumor (hopefully) detected.

There is no way to detect multiple images at once. If you have multiple images, you will have to upload them separately.

How did this project start?

This project started as a hackthon project. Within those days, my goal was to come up with a product to benefit the world as well as improve my backend, frontend, and machine learning skills. In the future, I hope to improve this project by adding a few more applications.
